# Hazard-Information-DA
위해정보 데이터분석보고서 공모전 협업 리포지토리 (서울시립대, 성균관대, 서울여대 학부생)

---



## 1. 배경 & 목적

- 프로젝트명 : 소비자 위해 정보 데이터 분석을 통한 소비자 안전사고 예방 방안 도출 모델 제작
- 목적 : 위해정보 데이터분석보고서 공모전 출전
- 배경 및 목표 : **소비자 위해 정보 데이터와 예측 모델을 활용하여 소비자 제품과 관련된 미래의 안전 위험을 예측하고, 완화하는 것 목표**로 공모전에 출전하게 되었다. 더 나아가 데이터 추세 파악을 기반으로 소비자 사고를 줄이기 위한 위해 사고 예측 기반의 검사 관리 시스템 구축하고자 한다.

## 4. 내용

1. 소비자 위해 정보 데이터 분석: 2020~2023년간의 데이터를 검토해서 제품 유형별로 발생하는 사고의 주요 원인을 파악한 뒤 특정 제품과 관련된 사고 빈도 및 특성의 추세 분석.
2. 고위험 제품 분류: 사고의 심각성과 빈도에 따라 제품 유형을 분류하고 데이터의 추세가 상승하는 제품 유형 파악.
3. 예측 모델 개발: 소비자 위해 정보 데이터 수집, 분석 이후 머신러닝 모델 훈련. 성별, 연령대, 사고 위치 등의 정보를 추가하여 정확도 개선. 시계열 분석을 통한 미래 위험 예측.
4. 외부적 요인: 기술 발전 요인, 규제 변경 관련 등 제품 안전에 영향을 미치는 요인 포함.
5. 예측 기반 검사 시스템 구축: 예측 모델의 결과를 바탕으로 고위험순으로 제품 카테고리를 식별하고 이에 따른 사고 예방 대책 구현.
